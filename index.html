Complaint Auto-Resolver

Follows your exact workflow: find open tickets, match them in the CMS dump by Charger ID + Name, and if the Connector is Active, close them out using its Last Heartbeat time.

  1. Open ticket = Complaint Resolve Date & Time is blank in the Main Tracker.
  2. Match that ticket in the CMS Dump by Charger ID and Charger Name.
  3. Check its Connector Status.
  4. If Active → copy Last Heartbeat into Resolve Date & Time, set Remarks = "Resolved".
