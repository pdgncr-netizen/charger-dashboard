🔗 Setup (one-time)

  1. Sheet 1 (Downtime): Google Sheet → File → Share → Publish to web → select tab → CSV → Publish → copy URL → paste above
  2. Sheet 2 (Complaints / Maintenance / Replacement): Same steps on new sheet. The Activity/Type column should contain Daily Complaint, Monthly Maintenance, or Replacement.
  3. Paste your Cloudflare Proxy URL (e.g. https://manish.pdgncr.workers.dev/) → click Refresh All Data

⚡ Hours column: dashboard auto-detects seconds vs hours. If values look wrong, check the Expected Power Down Time (HRS) column is actually in hours not seconds.
